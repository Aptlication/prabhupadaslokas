📄 SLOKA HUB — MASTER BUILD SPEC (FOR REPLIT)

🧭 PROJECT NAME

Sloka Hub

⸻

🎯 PROJECT PURPOSE

Build a web-based application (MVP) that helps ISKCON devotees:
	•	Learn slokas
	•	Improve pronunciation
	•	Understand meanings
	•	Memorize effectively

This is Phase 1 MVP only (no BBT content yet).

⸻

🔥 CORE PRODUCT IDEA

Transform users from repeating slokas → to understanding and confidently reciting them.

⸻

🧱 TECH STACK (REPLIT FRIENDLY)

Frontend:
	•	React (preferred) OR simple HTML/CSS/JS (if needed)
	•	Tailwind CSS (for clean UI)

Backend:
	•	Node.js (Express)

Storage:
	•	JSON (initial version)
	•	Later upgrade to database

⸻

📱 MVP FEATURES (PHASE 1 ONLY)

⸻

1. HOME SCREEN

Display:
	•	App name: Sloka Hub
	•	Tagline: Learn it. Live it. Lead it.

Buttons:
	•	Start Learning
	•	My Slokas
	•	Settings

⸻

2. SLOKA LIST SCREEN

Display list of slokas:

Each item shows:
	•	Sloka title (e.g., BG 2.13)
	•	First line (transliteration)

⸻

3. SLOKA DETAIL SCREEN

Each sloka contains:

A. Devanagari (optional for MVP)

B. Transliteration (main)

C. Word-by-word breakdown

⸻

4. TAP-TO-REVEAL WORD MEANING (CORE FEATURE)

Example:

aham → [tap] → "I"
sarvasya → [tap] → "of everything"

Behavior:
	•	Click word → show meaning below or flip card

⸻

5. AUDIO PLAYBACK

Feature:
	•	Play pronunciation audio
	•	Loop button
	•	Slow playback (optional)

⸻

6. REPETITION MODE

Buttons:
	•	Play full sloka
	•	Play line-by-line
	•	Repeat mode ON/OFF

⸻

7. SAVE TO “MY SLOKAS”

User can:
	•	Add sloka to personal list

⸻

8. SIMPLE PROGRESS TRACKING
	•	Mark sloka as:
	•	Learning
	•	Learned

⸻

🧠 DATA STRUCTURE (IMPORTANT)

{
